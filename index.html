About

Over the last 7 years, I have acquired a wide range of software development skills. I obtained a strong foundation in software development through my education at Sullivan University, and I have worked to continue building upon that knowledge with dedicated self-study and practical application both in the workplace and with personal projects.

While continuing to learn and improve competencies required to accomplish core responsibilities at Heartland Payment Systems and Stored Value Solutions, I have spent my free time researching JavaScript and responsive CSS to create useful, dynamic websites. I have also studied other programming languages including Java and C# to both hone skills I acquired from Sullivan University as well as broaden my experience with object-oriented programming and back-end frameworks.

Currently, I am participating in the Code Louisville training program to refresh my existing knowledge of web development technologies. Some of the content covered so far has been learning how to build a responsive website framework with HTML 5 and CSS3, discovering how to use JavaScript/jQuery to enhance page interactivity, and using AJAX to exchange data with web servers.

Experience

Service Delivery Engineer

Stored Value Solutions

September 2014 - Present | Louisville, KY

As a part of the IT Service Management team at Stored Value Solutions, I function as the ServiceNow Developer and Administrator. I partner with business leaders and process owners to architect and develop new applications, forms, and process flows for the company's ServiceNow platform. I maintain existing ServiceNow content and workflows, and I administer access within the site. I create custom JavaScript code that interfaces with ServiceNow's core processes to deliver enhancements to the site. I partner with end users to assist with issues that arise with ServiceNow.
